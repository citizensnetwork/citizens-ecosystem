# Feature Clarity

This folder holds "grill-me" questioning documents for features that need
design decisions before implementation.  Each MD covers one surface and is
intentionally exploratory — no decisions are locked until the session is
complete and a decision is logged in `.github/DECISIONS.md`.

| `search-and-discovery.md` | Search categories & contributor discovery | 🔴 Needs grill |
