# Citizens Connect

**Connecting the Kingdom** — A map-first community discovery platform that surfaces the vibrant layer of Christian community activity.

Built with Next.js 15 (App Router) + Supabase + Leaflet + Tailwind CSS v4 + Capacitor (iOS/Android).

## Quick Start

```bash
npm install
npm run dev
```
